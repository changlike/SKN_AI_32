# 자연어 텍스트 데이터 전처리 프로젝트 과제
- 프로젝트명 : konlpy_practice_project
뉴스 기사, PDF 텍스트, 크롤링 데이터 중 하나를 수집하여 다음 과정을 수행하시오.

### process
1. 텍스트 수집
2. 텍스트 정제
3. 형태소 분석
4. 불용어 제거
5. 단어 빈도 계산
6. PyTorch Tensor 변환
7. 워드클라우드 생성
8. 상위 20개 단어 시각화
9. 결과 분석 보고서 작성

### 프로젝트 간단 요약 
1. 데이터 출처: 뉴스 기사 (text.txt)
2. 처리 과정: 정제 -> Okt 명사 추출 -> 불용어 제거(186개 단어) -> 빈도 계산 -> Tensor 변환 -> 시각화
3. 상위 20개 단어: [('한국', 14), ('남아공', 10), ('승리', 6), ('월드컵', 6), ('가능성', 5), ('진출', 5), ('체코', 5), ('홍명보', 4), ('확률', 4), ('경기', 4), ('축구', 4), ('멕시코', 4), ('리그', 4), ('승점', 4), ('최종', 3), ('무승부', 3), ('대한민국', 2), ('통계', 2), ('매체', 2), ('감독', 2)]
4. 가장 많이 등장한 단어는 '한국'(14회)이다.
