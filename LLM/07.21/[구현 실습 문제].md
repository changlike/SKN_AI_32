## [구현 실습 문제]

 ## Prompt Engineering 실습 
 
 목표 같은 질문이라도 Prompt에 따라 결과가 어떻게 달라지는지 확인
 
  # 추가 메뉴 
  
  Prompt 실험 
  ├── 기본 Prompt 
  ├── 전문가 Prompt 
  ├── 친절한 설명 Prompt
   ├── 단계별 사고 Prompt
    └── JSON 출력 Prompt 
    
## 실습 내용 
    
    - Temperature 비교 
    - Top_p 비교 
    - System Prompt 비교 
    - Few-shot Prompt 
    
    Prompt를 직접 수정하여 결과를 확인하도록 합니다. 
    
    제공한 프로젝트에 위 실습내용을 프론트 페이지에 select 태그로 선택하게 해서 작동 확인하는 코드로 실행되도록 코드를 추가 작성하시오.

## 프론트 select 태그

Prompt 유형 선택 부분은 다음과 같습니다.

```html
<label>
    Prompt 유형

    <select id="promptType">
        <option value="basic">
            기본 Prompt
        </option>

        <option value="expert">
            전문가 Prompt
        </option>

        <option value="friendly">
            친절한 설명 Prompt
        </option>

        <option value="step_by_step">
            단계별 사고 Prompt
        </option>

        <option value="json">
            JSON 출력 Prompt
        </option>
    </select>
</label>
```

선택값이 변경되면 해당 Prompt의 기본 System Prompt와 수행 지시문이 자동으로 입력됩니다.

```javascript
promptType.addEventListener(
    "change",
    applySelectedPreset
);
```

```javascript
function applySelectedPreset() {
    const preset = promptPresets.find(
        (item) => item.value === promptType.value
    );

    if (!preset) {
        return;
    }

    systemPrompt.value = preset.system_prompt;
    customInstruction.value = preset.instruction;
}
```

자동 입력된 내용을 직접 수정한 뒤 다시 실행할 수 있습니다.
