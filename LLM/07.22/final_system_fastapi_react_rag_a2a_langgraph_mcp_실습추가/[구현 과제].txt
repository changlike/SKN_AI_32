[구현 과제]
완성 제공된 fastapi final project 에 이전 수업에 사용한 프로젝트 기능들을 연계하여 구동되게 합니다.

[구현 1]
사용자 문의에 '교환해 주세요', '환불해 주세요' 와 같은 실행 질문의 들어오면
LLM 은 '담당 부서로 연결해 드리겠습니다.' 로 답하게 하고,
MYSQL 의 fastapi_db 에  담당부서에 전달할 메세지로 등록 저장되게 하시오.

- 테이블명 : customer_complaint
- 컬럼 구성 : cc_id(구분아이디), custom_id(고객아이디), dept_id(담당부서아이디), message(내용),
                    receipt_status(접수여부), resolution_status(처리완료여부),
                    inquiry_date(문의날짜시간), receipt_date(접수날짜시간), resolution_date(처리완료시간날짜)
